2. Описание сервисов:
- Amazon ElastiCache:
Это AWS для кэширования, который совместим с популярными API-интерфейсами Redis и Memcached. Простыми словами, он ускоряет работу приложения за счет сохранения часто используемых данных в кэше + снижает нагрузку на БД, что делает наше приложение более производительным и масштабируемым. 
- Amazon ES
Это AWS для эффективного поиска, анализа и хранения больших объемов данных. То есть, если, например вам нужно как-то проанализировать все тиражи газет за последние 20 лет, вы можете собрать нужную информацию по ключевым словами через Elasticsearch почти мгновенно.
- Amazon QLDB
Это AWS база данных, с главным упором на прозрачность, неизменяемость данных, которая держится на криптографических протоколах. Обычно используется в приложениях, которые направлены на работу с финансами.
- Amazon KMS 
Это AWS для создания криптографических ключей. Условно, KMS шифрует и дешифрует твою БД, а также позволяет управлять жизнью этого ключа.
- Amazon Cloud HSM
Это AWS аппаратных модулей безопасности, предназначенных для надежного хранения твоих криптографических ключей в облаке. 	
- Amazon Rekognition
Это AWS предназначенный для распознавания/анализа большого количества изображений или видео. По сути это облачные алгоритмы CV для приложения.
- Amazon Textract
Это AWS предназначенный для распознавания/анализа большого количества текстовых данных. Грубо говоря это облачные алгоритмы NLP для приложения.
- Amazon Lex
Это AWS, который позволяет внедрять в ваше приложение чат-ботов или голосовых помощников. В его основе передовые алгоритмы машинного обучения, заточенные на распознавание речи и текстов.
- AWS Code Pipeline
Это AWS, который дает возможность настраивать пайплайны работы. Т.е. упрощает и автоматизирует процессы развертывания, тестирования и разработки программы. Проще говоря повышает эффективность работы всей команды разработчиков.
- Amazon SES
Это AWS электронной почты. Обеспечивает разработчикам надежную рассылку писем, защищенных от спама.
- Amazon SNS
Это AWS уведомлений, чем-то похожий на SES сервис, только он специализируется на отправке сообщений разных видов: уведомления, оповещения и т.п., а также делает это через различные каналы связи: SMS, HTTP протокол и другие.

**Описание отечественных аналогов:**
- Yandex Managed Service for Redis - делает  развертывания и управления кластерами Redis в облаке Яндекса
- Yandex Managed Service for Elasticsearch - предоставляет возможность развертывания и управления кластерами Elasticsearch в облаке Яндекса
- Yandex Key Management Service - дает возможность безопасного управления ключами шифрования и управление доступом к данным в облаке 
- Yandex Vision - встроенное CV
- Yandex SpeechKit - позволяет распознавать и анализировать речь и внедрять эту фичу в приложение.
- Yandex Cloud CI/CD - этот сервис упрощает и автоматизирует процессы развертывания, тестирования и разработки программы.
- Yandex Mail for Domain - это сервис для создания имейлов 
- Yandex Message Queue - большая по размаху и надежная пересылка сообщений
